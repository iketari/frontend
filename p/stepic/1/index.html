JavaScript - это не Java

Различия

Here are some differences between the two languages:

  • Java is a statically typed language; JavaScript is dynamic.
  • Java is class-based; JavaScript is prototype-based.
  • Java constructors are special functions that can only be called at object creation; JavaScript "constructors" are just standard functions.
  • Java requires all non-block statements to end with a semicolon; JavaScript inserts semicolons at the ends of certain lines.
  • Java uses block-based scoping; JavaScript uses function-based scoping.
  • Java has an implicit this scope for non-static methods, and implicit class scope; JavaScript has implicit global scope.

Документ на сервере

Схема (Браузер(Клик)) --> (Сетевой уровень(Запрос)) -- (Сервер(Выбор документ на сервере)) -> (Сетевой уровень((Ответ)) --> Браузер(Рендер документа))

Проблема: много однотипных документов, копипаста

Фронтенд: все, только верстка в ClientSide Бекенд: ничего.

Динамическая генерация документов

Схема (Клик) --> (Запрос) -- (Сервер, (Генерация документа)) -> (Ответ) --> (Рендер документа)

Проблема: генерация нового документа ради минимальных изменений, нагрузка на сервер

Фронтенд: обработка данных, серверная шаблонизация Бекенд: работа с БД, работа с файловой системой, сетевое взаимодействие с другими серверами, роутинг

Динамическое обновление документов

Схема (как на прошлом слайде) (Клик) --> (Запрос) -- (Сервер, (Генерация документа)) -> (Ответ) --> (Рендер документа) + без перезагрузки документа Схема (Клик) --> (AJAX запрос) -- Генерация фрагмента документа -> (Ответ) --> (Изменение исходного документа)

Проблема: много бизнес-логики на сервере, большие накладные издержки, нет структуры и снова копипаста

Транспорт данных

Схема (Клик) --> (AJAX запрос) -- Генерация данных -> (Ответ) --> (Рендер результата в браузере)

Gruntfile.js — Файлы***

  • * любые символы кроме / js/folder/*
  • ? любой 1 символ кроме / js/folder/file-?.js
  • ** любые символы, включая /, кроме имени файла js/**/*
  • {} разделенный через запятую список допустимых выражений js/folder{1,4,8}/*
  • ! в начале выражения, исключения из поиска ['js/**/*', '!js/old.js']

DOM API (validity)

  • customError: true (setCustomValidity())
  • patternMismatch: true (pattern)
  • rangeOverflow: true (max)
  • rangeUnderflow: true (min)
  • stepMismatch: true (step)
  • tooLong: true (maxLength)
  • typeMismatch: true (type)
  • valueMissing: true (required)
  • valid: true (all of the validity)
